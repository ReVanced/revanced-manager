# 💉 Patching apps

Learn how to use ReVanced Manager to patch apps.

## ✅ Steps to patch apps

1. Navigate to the `Apps` tab at the top navigation bar
2. Select an app to patch or tap select from storage[^1]
3. Optionally change the selection of patches[^1][^2]
4. Optionally select a specific downloader to download the app[^3]
5. Tap on the `Patch` button to begin the patching process
6. Follow the instructions on the screen if prompted
7. Tap on the `Install` button to install the patched app or export it with 💾 save button in bottom left corner

[^1]: This option is protected by safeguard
[^2]: It is recommended to use the default set of patches by tapping on the `Reset` button in the bottom right corner.
[^3]: By default, all available downloaders will be used to download the app.
If you want to use a specific downloader, you can change it here.
